# 						配置交换机的远程登录功能

1、配置交换机管理IP地址 

```c
Switch>enable           //进入特权模式
Switch#configure terminal					//进入全局配置 模式
Enter configuration commands, one per line.  End with CNTL/Z.
Switch(config)#interface vlan 1          //配置VLAN1
Switch(config-if)#ip address 192.168.1.1  255.255.255.0				//配置IP地址 和子网掩码
Switch(config-if)#no shutdown        //开启端口
Switch(config-if)#
%LINK-5-CHANGED: Interface Vlan1, changed state to up
Switch(config-if)#end
```

2、在交换机上配置远程用户登录验证密码

```c
Switch#configure  terminal
Enter configuration commands, one per line.  End with CNTL/Z.
Switch(config)#enable secret level 1 0 ruijie         //配置Telnet    即Telnet  1 密码
Switch(config)#enable secret level 15 0 ruijie        //配置Telnet    即enable  15 密码
Switch(config)#
```

3、在交换机 上启用远程登录 功能

```c
Switch(config)#line vty 0 4         //线程配置   同时可以有4个人访问
Switch(config-line)#password 0 ruijie      //配置Telnet  密码
Switch(config-line)#login				//启用Telnet用户密码进行验证
Switch(config-line)#exit
```

4、使用 ping命令测试网络的连通情况

```c
Switch#ping  192.168.1.1       	//ping一下自己 判断是否互通

Type escape sequence to abort.
Sending 5, 100-byte ICMP Echos to 192.168.1.1, timeout is 2 seconds:
!!!!!        //！  表示想通
Success rate is 100 percent (5/5), round-trip min/avg/max = 6/8/18 ms

Switch#
```

5、使用Telnet技术实现远程登录交换机

```c
telnet 192.168.1.1                 //在本机 上远程登陆交换机
Trying 192.168.1.1……
Password:ruijie					// Telnet密码
Switch>enable
Password:ruijie					// enable密码
Switch #
```
