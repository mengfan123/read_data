# 							配协议置交换机端口聚合

方法一

思科交换机，默认是开启交换机的生成树协议，要实现端口聚合，需先关闭生成树协议

```c
Switch-2(config)#no spanning-tree vlan 1         //关闭生成树协议
```

```c
Switch-3#configure terminal							//进入全局配置模式
Enter configuration commands, one per line.  End with CNTL/Z.
Switch-3(config)#no spanning-tree vlan 1				//关闭生成树协议
Switch-3(config)#interface port-channel 1				//配置聚合组一
Switch-3(config-if)#interface range fa 0/1-2			//将0/1和0/2接口分给第一组
Switch-3(config-if-range)#channel-group 1 mode on		//开通端口	
Switch-3(config-if-range)#end
```

```c
Switch-2#configure terminal									//进入全局配置模式
Switch-2(config)#no spanning-tree vlan 1					//关闭生成树协议
Switch-2(config)#interface port-channel 1					//配置聚合组一
Switch-2(config-if)#interface range fa0/1-2					//将0/1和0/2接口分给第一组
Switch-2(config-if-range)#channel-group 1 mode on			//开通端口	
Switch-2(config-if-range)#end
```



方法二

二层交换

```c
Switch(config)#no spanning-tree vlan 1				//关闭生成树协议
Switch(config)#interface range fa0/1-2				//进入子接口模式
Switch(config-if-range)#channel-group 1 mode desirable		//创建组一
Switch(config-if-range)#switchport mode trunk				//配置端口模式
Switch(config-if-range)#no shutdown 						//激活端口
Switch(config-if-range)#
```

三层交换

```c
Switch(config)#no spanning-tree vlan 1				//关闭生成树协议
Switch(config)#interface range fa 0/1-2					//进入子接口模式
Switch(config-if-range)#channel-group 1 mode desirable		//创建组一
Switch(config-if-range)#switchport trunk encapsulation dot1q //配置trunk协议
Switch(config-if-range)#switchport mode trunk				//配置端口模式
Switch(config-if-range)#no shutdown 						//激活端口
```
